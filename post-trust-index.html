Introducing the Fabric Trust Index ‚Äî free public scores on 10,000+ AI services

Today we're launching the Fabric Trust Index: a free, searchable database of trust scores for every AI service, model, and MCP skill we can find. Open access. No account required. Updated continuously from public data sources.

Why we built this

The AI ecosystem is scaling faster than anyone can audit. In the past 12 months, the number of MCP servers published across npm, PyPI, and dedicated registries has gone from a few hundred to tens of thousands. Models ship weekly. New AI tools and APIs appear daily. And for developers and businesses building agent-based systems, the question isn't whether to use third-party AI services ‚Äî it's which ones are safe to use.

Right now, the answer is mostly vibes. Download counts. GitHub stars. A README that looks professional. Maybe a friend on Twitter vouched for it. None of these are trust signals ‚Äî they're popularity signals, and they're trivially gamed. The ClawHub malware incident proved that the #1 most-downloaded tool on a platform can be actively stealing your credentials.

We think the ecosystem deserves better. So we built it.

What the Trust Index does

Every service in the index receives a trust score from 0.00 to 5.00, computed automatically from six independent signals: vulnerability and safety analysis, operational health monitoring, maintenance activity, adoption metrics, transparency checks, and publisher trust verification. The signals are weighted by risk ‚Äî security carries more weight than popularity ‚Äî and the combination makes gaming the system extremely difficult.

Coverage at launch

The initial index includes over 10,000 services scraped and scored from npm, PyPI, HuggingFace, and major MCP registries. Coverage skews toward MCP tools and AI APIs ‚Äî the categories where trust matters most and where the scoring signals are richest. We're adding new registries and service types continuously.

If a service you're looking for isn't in the index yet, searching for it triggers an on-demand scoring run. Most services are indexed and scored within minutes of the first search.

Open by default

The Trust Index is free and doesn't require an account. We believe trust data should be a public good ‚Äî not locked behind a paywall. If you're a developer evaluating whether to add an MCP server to your agent, you should be able to check its score in 10 seconds without signing up for anything.

Free access includes:

Unlimited searches, full score breakdowns, per-signal detail, historical trends, and category browsing. No rate limits on the web interface. The Trust Index is and will remain free.

For providers

If you build or maintain an AI service, your score is computed automatically from public data ‚Äî no action required. But if you want to claim your profile, add context, or dispute a score, we've built a provider portal for that. Claiming a profile is free and gives you access to detailed scoring rationale, improvement suggestions, and a verified badge once ownership is confirmed.

Providers cannot pay for a higher score. The only way to improve a score is to improve the underlying signals: patch vulnerabilities, maintain uptime, respond to issues, publish documentation, and build from a verified identity. Trust is earned, not purchased.

The API

The Trust Index is designed to be searched by humans. But the real value unlocks when agents can query it programmatically. The Fabric API lets you gate every tool call, every service integration, and every MCP connection behind a trust check ‚Äî automatically, in real time, before the agent acts.

Free 50 requests/day. Search scores, view breakdowns. Perfect for individual developers and evaluation.
Pro 15,000 requests/day. Custom trust thresholds, webhook alerts on score changes, priority re-scoring.
Team 50,000 requests/day. Organisation-wide policies, audit logs, allowlists/blocklists, custom signal weights.

What's next

This is v1. We're shipping fast and iterating based on what the ecosystem needs. Near-term priorities include on-chain attestation of scores via Base, expanded registry coverage, a browser extension for inline scoring on npm and GitHub, and deeper integration patterns for popular agent frameworks like LangChain, CrewAI, and OpenClaw.

If you have feedback, feature requests, or want to integrate ‚Äî we're listening. Reach out on GitHub or Twitter / X.
